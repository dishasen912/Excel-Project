# Excel-Project

## Sales Performance Dashboard

Yeh project Microsoft Excel me banaya gaya **Sales Performance Dashboard** hai, jo sales data ko analyze aur visualize karne ke liye use hota hai.

## Project Description
Is dashboard me sales data ko charts aur tables ke through represent kiya gaya hai, jisse business performance ko easily samjha ja sake.  
Yeh project Excel ke basic aur intermediate features par based hai.

## Features
- Total Sales analysis  
- Monthly / Yearly sales overview  
- Charts aur graphical representation  
- Easy to understand dashboard layout  

## Files Included
- `sales performance dashboard.xlsx` – Main Excel dashboard file  
- `README.md` – Project documentation  

## Tools Used
- Microsoft Excel  

## How to Use
1. Repository ko download ya clone karein  
2. `sales performance dashboard.xlsx` file open karein  
3. Data aur dashboard ko explore karein

# Excel-Project

## Sales Performance Dashboard

This project is a **Sales Performance Dashboard** developed using **Microsoft Excel**.  
It helps analyze sales data and presents key business insights through interactive dashboards, charts, and tables.

---

## Project Overview

The Sales Performance Dashboard is designed to track and evaluate sales performance over different time periods.  
It provides a clear view of total sales, monthly and yearly trends, and overall business performance using Excelâ€™s visualization tools.

This project demonstrates **basic to intermediate Excel skills**, making it suitable for learning, practice, and portfolio use.

## Project Description

The dashboard represents raw sales data in a structured and visual format using:
- Tables for organized data
- Charts for trend analysis
- Summary metrics for quick insights

This helps businesses and users:
- Monitor sales growth
- Identify trends and patterns
- Make data-driven decisions
